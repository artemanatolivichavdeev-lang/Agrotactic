## О проекте

Корпоративный сайт-витрина продукта **«Агротактик»** — операционного контура для управления агрокомплексами. Сайт рассчитан на enterprise-аудиторию (агрохолдинги, крупные хозяйства) и работает как инструмент для прохождения внутренних согласований пилота/внедрения.

Весь код разрабатывается в Google Antigravity, хранится в этом репозитории, а публикуется через механизмы Tilda (Custom CSS, HEAD-код, блоки T123 «Embed HTML Code»).

## Что показывает продукт

«Агротактик» — операционный контур / оркестратор:

- **Диспетчеризация и алерты** — очередь отклонений, приоритеты, ответственность
- **План-факт и KPI** — наблюдаемость в реальном времени
- **Интеграции** поверх существующих систем (ERP/1С, телематика, датчики, BI, API)
- **Журнал действий / Audit trail** — кто, что, когда, почему
- **Комплаенс и отчётность** как часть процесса (включая «Меркурий»)
- **Лестница автономии** — поэтапное увеличение: A2 → A3 → A4

### Изоляция стилей

Весь пользовательский код изолирован от стилей Tilda:
- Обёртка `.x-site` для всего контента
- Все CSS-классы с префиксом `x-*`
- Глобальные селекторы (`body`, `h1`, `a`) не используются без `.x-site`

## Публикация на Tilda

Подробная инструкция — в файле [`DEPLOY_TILDA.md`](tilda/DEPLOY_TILDA.md).

Краткий порядок:

1. **Custom CSS** → Site Settings → More → Custom CSS → вставить `CUSTOM_CSS.css`
2. **HEAD** → Page Settings → Additional → HTML in HEAD → вставить `SITE_HEAD.html`
3. **8 блоков T123** → добавить блоки Other → T123, вставить файлы `T123_01` — `T123_08` по порядку
4. **Publish** → опубликовать страницу + «Publish all pages»

> ⚠️ Блоки T123 выполняются **только** на опубликованной странице. В превью Tilda код может не работать.
