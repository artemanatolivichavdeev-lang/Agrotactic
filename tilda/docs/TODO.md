# Post-Deployment Checklist (v0.1)

Этот файл поможет "довести" сайт после вставки кода в Tilda.

## 1. Якоря (Anchors)

- [ ] Опубликовать страницу.
- [ ] Найти ID блока формы (напр. `#rec12345678`).
- [ ] Найти ID блока контактов (напр. `#rec87654321`).
- [ ] Заменить `#recFORM` и `#recCONTACTS` в коде `home.html` (или через "Search & Replace" в вебе).

## 2. Документы (PDFs)

- [ ] Загрузить файлы в Tilda ("Files" или "Upload").
- [ ] Получить прямые ссылки (URL).
- [ ] Вставить ссылки в Drawer вместо `javascript:void(0)` (или настроить отдельный JS-маппинг).

## 3. SEO & Analytics

- [ ] Прописать meta title / description в настройках страницы Tilda.
- [ ] Включить ленивую загрузку (Lazy Load) для изображений в `site-head.html` (если нужно).
- [ ] Проверить Google Analytics / Яндекс.Метрику.

## 4. Тесты

- [ ] Отправить тестовую заявку.
- [ ] Проверить все ссылки в футере.
- [ ] Проверить мобильную версию на реальном устройстве (iOS / Android).
