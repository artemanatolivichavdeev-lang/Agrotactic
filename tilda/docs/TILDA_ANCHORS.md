# TILDA_ANCHORS (Mapping)

Файл `home.html` использует эти placeholder-якоря.
Замени их на реальные ID блоков Tilda (из Settings блока или URL #rec...).

| В коде (Placeholder) | В Tilda (Real ID) | Назначение                    |
| -------------------- | ----------------- | ----------------------------- |
| `#recFORM`           | `#rec__________`  | Основная форма (Lead Capture) |
| `#recCONTACTS`       | `#rec__________`  | Блок контактов (Footer)       |
| `#x-model`           | (внутри кода)     | Якорь секции "Модель"         |
| `#x-platform`        | (внутри кода)     | Якорь секции "Платформа"      |
| `#x-integrations`    | (внутри кода)     | Якорь секции "Интеграции"     |
| `#x-pilot`           | (внутри кода)     | Якорь секции "Пилот"          |

> **Инструкция:**
>
> 1. Опубликуй страницу в Tilda.
> 2. Найди реальные ID блоков формы и контактов.
> 3. Вставь их в таблицу выше.
> 4. Замени в `tilda/t123/home/home.html` (Ctrl+H).
